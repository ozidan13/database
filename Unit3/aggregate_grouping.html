Aggregation

دوال التجميع والتصنيف

عايز تعرف إجمالي المبيعات؟ متوسط الراتب؟ عدد الموظفين في كل قسم؟ الـ Aggregation هو الجواب! 📊

دوال التجميع الأساسية

Aggregate Functions

COUNT()

بتعدّ عدد الصفوف. COUNT(*) بتعدّ الكل، COUNT(col) بتتجاهل الـ NULL.

SUM()

مجموع قيم عمود رقمي. مثال: إجمالي الرواتب.

AVG()

المتوسط الحسابي. بتتجاهل الـ NULL تلقائياً.

MAX()

أكبر قيمة في العمود. بتشتغل مع الأرقام والتواريخ والنصوص.

MIN()

أصغر قيمة في العمود.

Aggregate Functions
-- إحصاءات عامة عن الموظفين SELECT COUNT(*) AS total_employees, COUNT(dept_id) AS assigned_employees, -- يتجاهل NULL SUM(salary) AS total_payroll, AVG(salary) AS avg_salary, MAX(salary) AS highest_salary, MIN(salary) AS lowest_salary FROM employee;

GROUP BY — التجميع حسب فئة

Grouping Records

GROUP BY بتقسّم الصفوف لمجموعات حسب قيمة عمود — وبعدين بنطبق دوال التجميع على كل مجموعة.

مثال: عايز متوسط الراتب لكل قسم — يعني بنجمّع الموظفين حسب الـ dept_id.

GROUP BY Examples
-- عدد الموظفين وإجمالي الرواتب لكل قسم SELECT dept_id, COUNT(*) AS emp_count, SUM(salary) AS total_salary, ROUND(AVG(salary), 2) AS avg_salary FROM employee GROUP BY dept_id ORDER BY total_salary DESC; -- عدد الموظفين حسب تاريخ التعيين (بالسنة) SELECT EXTRACT(YEAR FROM hire_date) AS hire_year, COUNT(*) AS hired_count FROM employee GROUP BY hire_year ORDER BY hire_year; -- GROUP BY على أكتر من عمود SELECT dept_id, is_active, COUNT(*) AS count FROM employee GROUP BY dept_id, is_active;

HAVING — فلترة المجموعات

Filtering Groups (Not Rows!)

HAVING ≠ WHERE!

WHERE

بيفلتر الصفوف الفردية قبل الـ GROUP BY.

HAVING

بيفلتر المجموعات بعد الـ GROUP BY. بتقدر تستخدم دوال التجميع.

HAVING + WHERE + GROUP BY
-- الأقسام اللي عدد موظفيها أكتر من 3 SELECT dept_id, COUNT(*) AS emp_count FROM employee GROUP BY dept_id HAVING COUNT(*) > 3; -- WHERE + GROUP BY + HAVING معاً SELECT dept_id, AVG(salary) AS avg_sal FROM employee WHERE is_active = TRUE -- فلتر الصفوف أول GROUP BY dept_id -- بعدين جمّع HAVING AVG(salary) > 7000 -- بعدين فلتر المجموعات ORDER BY avg_sal DESC;

Quick Knowledge Check

❓ سؤال ١: عايز أعرف عدد الموظفين في كل قسم — إيه الأمر؟

SELECT dept_id, COUNT(*) FROM employee;
SELECT dept_id, COUNT(*) FROM employee GROUP BY dept_id;
SELECT dept_id, COUNT(*) FROM employee HAVING dept_id;

❓ سؤال ٢: ما الفرق بين WHERE و HAVING؟

مفيش فرق — استخدم أي واحد
WHERE بيفلتر الصفوف قبل GROUP BY — HAVING بيفلتر المجموعات بعده
WHERE للأرقام — HAVING للنصوص

❓ سؤال ٣: COUNT(*) و COUNT(column) بيختلفوا إزاي؟

مفيش فرق
COUNT(*) بيعدّ كل الصفوف — COUNT(col) بيتجاهل الـ NULL
COUNT(*) بطيء — COUNT(col) أسرع دايماً
