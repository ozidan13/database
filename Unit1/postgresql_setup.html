Why We Chose PostgreSQL

في كذا Relational Database ممكن نستخدمها — بس إحنا اخترنا PostgreSQL لأسباب كتير:

🆓

مجاني ومفتوح المصدر

مش محتاج تدفع حاجة — على عكس Oracle اللي ممكن يكلفك آلاف الدولارات.

🏢

بيتستخدم في الشركات الكبيرة

Apple, Instagram, Spotify, Uber — كلهم بيستخدموا PostgreSQL.

📚

متوافق مع المنهج الأكاديمي

بيدعم كل حاجة في كتاب Elmasri & Navathe — Triggers, Views, Constraints... كل حاجة!

⚡

فيه Features متقدمة

بيدعم JSON, Full-Text Search, Window Functions, CTEs — حاجات MySQL لسه بيلحق فيها.

🔌

بيشتغل مع Prisma

في الوحدات المتقدمة هنستخدم Prisma ORM مع NestJS — و PostgreSQL هو الأفضل معاهم.

🌍

مجتمع ضخم

أي مشكلة هتقابلك — هتلاقي حلها على Stack Overflow في ثواني.

خطوات التثبيت

Step-by-Step Installation Guide

Windows
macOS
Linux

Download the Installer

روح على الموقع الرسمي: postgresql.org/download/windows — ونزّل الـ Windows x86-64 Installer (أحدث إصدار).

Run the Installer

شغّل الملف اللي نزلته → Next → اختار مكان التثبيت (سيبه Default) → اختار كل الـ Components (خاصة pgAdmin 4 و Command Line Tools).

Set Superuser Password

مهم جداً! هيطلب منك تعمل باسورد للمستخدم postgres — ده الـ superuser. احفظ الباسورد ده كويس! لو نسيته هتضطر تعيد التثبيت. 🔐

Port Configuration

الـ Port الافتراضي هو 5432 — سيبه زي ما هو إلا لو عندك حاجة تانية شغالة عليه.

Verify Installation

بعد ما يخلص — افتح Command Prompt أو PowerShell واكتب:
PowerShell
PS> psql --version
psql (PostgreSQL) 16.2
لو ظهرت النسخة — مبروك! 🎉 التثبيت تمام. لو مطلعش — لازم تضيف PostgreSQL bin folder للـ PATH.

Install via Homebrew

أسهل طريقة على Mac — استخدم Homebrew:
Terminal
$ brew install postgresql@16
==> Installing postgresql@16...
✓ postgresql@16 was successfully installed!

Start the Service

Terminal
$ brew services start postgresql@16
==> Successfully started `postgresql@16`

Verify

Terminal
$ psql --version
psql (PostgreSQL) 16.2

Install via apt (Ubuntu/Debian)

Terminal
$ sudo apt update && sudo apt install postgresql postgresql-contrib
Reading package lists... Done
Setting up postgresql-16 (16.2-1) ...

Start & Enable

Terminal
$ sudo systemctl start postgresql
$ sudo systemctl enable postgresql
Created symlink...

Set Password

Terminal
$ sudo -u postgres psql
postgres=# ALTER USER postgres PASSWORD 'your_password';
ALTER ROLE

أول Query!

Your First PostgreSQL Commands

تمام — PostgreSQL اتثبت! دلوقتي خلينا نتصل بيها ونكتب أول أوامر. 🎉

الاتصال بالـ PostgreSQL

افتح الـ Terminal واكتب الأمر ده عشان تتصل:
psql
$ psql -U postgres
Password for user postgres: ********
postgres=# ← أنت جوا PostgreSQL دلوقتي! 🎉

إنشاء أول Database

psql — postgres
postgres=# CREATE DATABASE mastery_db;
CREATE DATABASE

postgres=# \c mastery_db
You are now connected to database "mastery_db".

إنشاء أول Table وإدخال بيانات

psql — mastery_db
mastery_db=# CREATE TABLE students (
id SERIAL PRIMARY KEY,
name VARCHAR(100) NOT NULL,
gpa DECIMAL(3,2)
);
CREATE TABLE

mastery_db=# INSERT INTO students (name, gpa) VALUES
('أحمد محمد', 3.5),
('فاطمة علي', 3.8),
('عمر حسن', 3.2);
INSERT 0 3

mastery_db=# SELECT * FROM students;
id | name | gpa
----+-------------+------
1 | أحمد محمد | 3.50
2 | فاطمة علي | 3.80
3 | عمر حسن | 3.20
🎉 مبروك! إنت لسه كتبت أول CREATE TABLE، INSERT، و SELECT — ده الأساس بتاع كل حاجة هنتعلمها!

pgAdmin 4 — الواجهة الرسومية

The GUI Alternative

مش لازم تشتغل بالـ Terminal على طول — في أداة رسمية اسمها pgAdmin 4 بتيجي مع PostgreSQL:

① افتح pgAdmin 4 من قائمة البرامج — هيفتح في المتصفح.

② ادخل الباسورد بتاع المستخدم postgres.

③ هتلاقي الـ Databases في الشمال — اضغط على mastery_db.

④ اضغط على Query Tool في الأعلى — وابدأ اكتب SQL!

💡 pgAdmin حلوة للمبتدئين عشان بتشوف البيانات في جدول. بس في الكورس ده هنستخدم الـ Terminal أكتر — لأنه أسرع وبيبني عضلات! 💪
