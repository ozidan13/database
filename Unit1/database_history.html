A Journey Through Database History

🗓️ 1960s — البداية

Flat Files & File Systems

كل حاجة كانت بتتخزن في ملفات مسطحة (Flat Files). كل برنامج كان بيدير ملفاته بنفسه. مفيش معايير، مفيش مشاركة — فوضى! أول محاولات لتنظيم البيانات ظهرت في IBM ووكالة الفضاء NASA.
🗓️ 1966-1969 — أول أنظمة حقيقية

IMS & CODASYL (Hierarchical & Network)

IBM بنت IMS لمشروع أبولو الفضائي 🚀 — أول DBMS حقيقي! كان بيخزن البيانات في شكل شجرة (هيراركي). بعده ظهر نموذج الشبكة (Network Model - CODASYL) اللي سمح بعلاقات أكتر مرونة.
🏢 IBM — IMS (1966)
🗓️ 1970 — نقطة التحول! 🔥

Relational Model — Edgar F. Codd

أهم لحظة في تاريخ قواعد البيانات! الدكتور Edgar F. Codd في IBM Research نشر ورقة بحثية بعنوان "A Relational Model of Data for Large Shared Data Banks". اقترح إن البيانات تتخزن في Tables (جداول) مترابطة — وده أساس كل الـ Relational Databases اللي بنستخدمها اليوم!
👤 Edgar F. Codd — IBM Research
🗓️ 1974-1979 — SQL وأول RDBMS

System R, SEQUEL → SQL, Oracle

IBM بنت System R — أول تطبيق عملي للنموذج العلائقي. معاه اتطورت لغة SEQUEL اللي بقت SQL! في ١٩٧٩، Larry Ellison أسس Oracle — أول RDBMS تجاري ناجح. 💰
👤 Larry Ellison — Oracle (1979)
🗓️ 1980s-1995 — العصر الذهبي

SQL Standard & PostgreSQL Origins

SQL بقت معيار عالمي (ANSI SQL 1986). ظهرت DB2 من IBM، و SQL Server من Microsoft. في ١٩٨٦، Michael Stonebraker في UC Berkeley بدأ مشروع POSTGRES — اللي بقى PostgreSQL اللي بنستخدمه النهاردة! 🐘
👤 Michael Stonebraker — PostgreSQL
🗓️ 2004-2010 — ثورة NoSQL

NoSQL Revolution

مع ظهور Web 2.0 والـ Big Data، الشركات زي Google, Facebook, Amazon احتاجت حلول جديدة. Google نشرت BigTable، Amazon نشرت Dynamo — ومنهم ظهرت MongoDB, Cassandra, Redis. مصطلح "NoSQL" بقى ترند! 🔥
🗓️ 2010s — الحاضر والمستقبل

NewSQL, Cloud & AI Databases

ظهرت الـ Cloud Databases (AWS RDS, Google Cloud SQL, Supabase). والـ NewSQL (CockroachDB, TiDB) بتجمع بين قوة SQL ومرونة NoSQL. والـ Vector Databases (Pinecone, Weaviate) ظهرت للـ AI والـ LLMs! 🤖

The Pioneers You Should Know

🏆 Edgar F. Codd

أبو النموذج العلائقي. حاصل على Turing Award 1981. من غيره مكانش في SQL ولا Relational Databases. ورقته في ١٩٧٠ غيرت عالم البيانات للأبد.

🏆 Michael Stonebraker

مؤسس POSTGRES (أصل PostgreSQL). حاصل على Turing Award 2014. ساهم في تطوير مفاهيم الـ Object-Relational الموجودة في PostgreSQL النهاردة.

💼 Larry Ellison

مؤسس Oracle Corporation. أخذ ورقة Codd وحولها لمنتج تجاري ناجح. النهاردة Oracle من أكبر شركات البرمجيات في العالم. 💰
