The Major Database Types

Relational (SQL)

قواعد البيانات العلائقية
البيانات بتتخزن في Tables (جداول) مترابطة مع بعض بالـ Keys. ده أشهر نوع وأقدمه — وبيستخدم لغة SQL للاستعلامات. مثالي للبيانات المنظمة اللي ليها بنية ثابتة.
PostgreSQL MySQL Oracle SQL Server SQLite

Document Store

قواعد بيانات المستندات
البيانات بتتخزن كمستندات JSON مرنة. مش لازم كل مستند يبقى ليه نفس البنية. مثالي للتطبيقات اللي بنيتها بتتغير كتير وللـ APIs.
MongoDB CouchDB Firestore

Key-Value Store

قواعد بيانات المفتاح/القيمة
أبسط نوع — كل حاجة بتتخزن كـ Key وقيمة مرتبطة بيه. زي الـ Dictionary في البرمجة. سريع جداً — بيتستخدم في Caching والـ Sessions.
Redis DynamoDB Memcached

Graph Database

قواعد بيانات الرسم البياني
البيانات بتتخزن كـ Nodes (عُقد) و Edges (علاقات). مثالي لما العلاقات بين البيانات أهم من البيانات نفسها — زي شبكات التواصل الاجتماعي. 🕸️
Neo4j ArangoDB Amazon Neptune

Column-Family

قواعد بيانات الأعمدة
بتخزن البيانات بالأعمدة مش بالصفوف. مثالي للـ Big Data والتحليلات — لما عايز تقرأ عمود كامل (مثلاً كل الأسعار) مش صف كامل.
Cassandra HBase ScyllaDB

Time-Series

قواعد البيانات الزمنية
متخصصة في بيانات مرتبطة بالزمن — زي قراءات السنسورات، أسعار الأسهم، logs السيرفرات. بتضغط البيانات بكفاءة وبتستعلم بسرعة على نطاقات زمنية.
InfluxDB TimescaleDB Prometheus

🤔 طيب... أستخدم إيه؟

بيانات منظمة + علاقات كتير + محتاج ACID؟ → Relational (PostgreSQL)

بيانات مرنة + API سريعة + مش محتاج Joins كتير؟ → Document (MongoDB)

محتاج سرعة رهيبة + بيانات بسيطة؟ → Key-Value (Redis)

العلاقات بين البيانات هي الأهم؟ → Graph (Neo4j)

Quick Knowledge Check

❓ سؤال ١: لو عايز تبني تطبيق بنكي بيحول فلوس — إيه أنسب نوع Database؟

MongoDB — لأنها سريعة
PostgreSQL — لأنها Relational وبتدعم ACID
Redis — لأنها أسرع حاجة
Neo4j — لأنها بتدعم علاقات

❓ سؤال ٢: لو عايز تخزن بيانات بوستات بشكل مرن من غير Schema ثابت — إيه أنسب نوع؟

Oracle — أقوى Relational
Cassandra — للبيانات الكبيرة
MongoDB — Document Store مرن
InfluxDB — للبيانات الزمنية

الخلاصة

Key Takeaways

① الـ Relational Databases هي الأكثر انتشاراً — وهنركز عليها في الكورس ده (خاصة PostgreSQL).

② الـ NoSQL مش "أحسن" من SQL — كل نوع ليه استخدامه.

③ الشركات الكبيرة بتستخدم أكتر من نوع — ده اسمه Polyglot Persistence.

④ في الدرس الجاي هنفهم معمارية الـ Database من جوا — الـ Three-Schema Architecture! 🏗️
